Frequently Asked Questions

Quick answers to common questions about using GateFlux.

For Residents

How do I pre-approve a visitor?
  1. Open the GateFlux app and tap Add Visitor
  2. Enter the visitor's name and phone number
  3. Choose a visit date and expected time
  4. Tap Create to generate a QR code

The visitor will receive an SMS with entry instructions. Learn more

Can I let in a delivery without pre-approving?

Yes! When the delivery person arrives, the security guard will send you a notification requesting approval. You can approve or deny directly from the notification or the app.

How do I set up recurring access for my maid or cook?
  1. Go to Visitors → Recurring
  2. Tap Add Recurring Visitor
  3. Enter the provider's details
  4. Set the schedule (e.g., Mon-Sat, 8 AM - 10 AM)
  5. The provider can now enter during those times without daily approval
I'm not receiving notifications. What should I do?
  1. Check that notifications are enabled for GateFlux in your phone settings
  2. Make sure you're logged in to the app
  3. Check if Do Not Disturb mode is off
  4. Try logging out and logging back in

If the problem continues, contact your society admin.

Can I manage visitors for multiple flats?

Yes! If you're registered for multiple flats (e.g., you own A-101 and B-202), you can switch between them using the flat selector at the top of the app. Each flat has its own visitor list and settings.

How do I cancel a pre-approved visitor?
  1. Go to Visitors → Expected
  2. Find the visitor you want to cancel
  3. Tap on their entry and select Cancel

The visitor will be notified that their access has been revoked.

Can my family members also approve visitors?

Yes! Your society admin can add multiple family members to the same flat. Each person logs in with their own phone number and can approve visitors independently.

For Security Guards

A visitor doesn't have a QR code. What do I do?
  1. Search for the visitor's name or phone in the Expected Visitors list
  2. If found, verify their ID and check them in manually
  3. If not found, register them as a Walk-in and request approval from the resident
How do I check out a visitor?
  1. Go to Currently Inside
  2. Find the visitor (search by name or flat)
  3. Tap Check Out

The visit time is automatically logged.

What if the resident doesn't respond to an approval request?
  • Wait for the timeout period (usually 5 minutes)
  • If urgent, try calling the resident using the contact button in the app
  • If no response, politely ask the visitor to wait or come back later
  • Do not allow entry without approval (unless it's an emergency)
How do I handle an emergency situation?

For emergencies (ambulance, fire, police):

  1. Allow immediate entry
  2. Use Walk-in → Emergency Entry to log the visit
  3. Select the emergency type
  4. Notify your supervisor
The app says a visitor is "overstaying." What does that mean?

Overstaying means the visitor has been inside for longer than the expected duration. You should:

  1. Contact the resident to verify the situation
  2. If the visitor has left, check them out to update the records
  3. If still visiting, no action needed (it's just an alert)

Account & App

I didn't receive the OTP. What should I do?
  1. Wait 60 seconds (OTPs can be delayed)
  2. Tap "Resend OTP"
  3. Make sure you entered the correct phone number
  4. Check that your phone has network signal
  5. If still not received, contact your society admin
I moved to a new flat. How do I update my account?

Contact your society admin. They will:

  1. Remove you from your old flat
  2. Add you to your new flat

Your phone number stays the same, so you can continue using the same login.

Can I change my phone number?

Yes, but you'll need help from your society admin. Contact them with your old and new phone number, and they'll update your account.

Is my data safe and private?
  • All data is encrypted in transit and at rest
  • Only your society admin and security can see visitor records for your flat
  • Other residents cannot see your visitor history
  • Visitor photos are stored securely and access is restricted
The app is slow or not loading. What should I do?
  1. Check your internet connection
  2. Force close and reopen the app
  3. Update to the latest version from the app store
  4. Restart your phone if the problem persists

Security & Privacy

What is Two-Factor Authentication (2FA)?

Two-Factor Authentication adds an extra layer of security to your account. After entering your password, you'll need to provide a 6-digit code from an authenticator app (like Google Authenticator or Microsoft Authenticator) to log in.

How do I enable Two-Factor Authentication?
  1. Go to Profile → Security → Two-Factor Authentication
  2. Click Enable 2FA
  3. Scan the QR code with your authenticator app
  4. Enter the 6-digit code from the app to confirm
  5. Save your recovery codes in a secure location
What if I lose my 2FA device or can't access my authenticator app?

Use one of your recovery codes that you saved when setting up 2FA:

  1. On the login screen, click Use Recovery Code
  2. Enter one of your 8 recovery codes
  3. Each recovery code can only be used once

If you've used all recovery codes, contact your administrator to reset 2FA.

How do I regenerate my recovery codes?
  1. Go to Profile → Security → Two-Factor Authentication
  2. Click Regenerate Recovery Codes
  3. Save the new codes securely

Note: Old recovery codes will no longer work after regenerating.

Can I disable Two-Factor Authentication?

Yes, unless your organization requires 2FA:

  1. Go to Profile → Security → Two-Factor Authentication
  2. Click Disable 2FA
  3. Confirm by entering your current 2FA code

Warning: If your platform requires 2FA for administrators, you may need to re-enable it to access admin features.

Which authenticator apps are supported?

GateFlux uses standard TOTP (Time-based One-Time Password), compatible with any authenticator app:

  • Google Authenticator
  • Microsoft Authenticator
  • Authy
  • 1Password
  • Any other TOTP-compatible app

For Admins

How do I add a new resident?
  1. Go to Residents in the admin panel
  2. Click Add Resident
  3. Enter their name, phone number, and flat number
  4. Click Save

The resident can now log in using their phone number. Learn more

Can I import residents from a spreadsheet?

Yes! Use the bulk import feature:

  1. Go to Residents → Import
  2. Download the template CSV file
  3. Fill in your resident data
  4. Upload the completed file
How do I remove a resident who moved out?
  1. Go to Residents
  2. Search for the resident
  3. Click on their profile
  4. Click Deactivate (this preserves history) or Delete (permanent)

Deactivating is recommended so you keep visitor records for that flat.

How do I view visitor analytics?

Go to Analytics in the admin panel. You can see:

  • Daily/weekly/monthly visitor counts
  • Peak visiting hours
  • Visitor types breakdown (guests, delivery, service)
  • Average visit duration

Export reports as CSV or PDF for record-keeping.

How do I require Two-Factor Authentication for all admins?

Platform administrators can enforce 2FA for all admin users:

  1. Go to Platform Settings → Security
  2. Enable Require Two-Factor Authentication
  3. Save changes

Once enabled, all admins must set up 2FA before accessing administrative features.

How do I reset a user's Two-Factor Authentication?

If a user is locked out and has used all recovery codes:

  1. Go to Users in the admin panel
  2. Find and select the user
  3. Click Reset 2FA
  4. The user can now log in and set up 2FA again

Only perform this after verifying the user's identity through other means.
